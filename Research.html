Masters Thesis:

Reactionless Visual Servoing of Floating Base Dual-Arm Manipulator

Reactionless Visual Servoing of Space Manipulator.

    Control of a fixed based manipulator is a well addressed problem. A floating base adds additional degree of difficulty to control problem. Visual seroving of a dual-arm manipulator mounted on a floating base is addressed in this thesis work. The following are the highlights of the work:

  • Formulation of Generalized Jacobian Matrix to control the end-effectors while accounting for the base disturbance, similar to motion rate control(using jacobian) of fixed based manipulators .
  • Visual servoing of the dual arm manipulator while using the above control scheme.
  • The system is a redundant system. Task priority formulation was used to project a secondary task onto the nullspace of the primary task(Visual servoing).
  • Secondary task was formulated as a minimization of cost function for attitude disturbance. This way, we could reduce the base attitude disturbance to zero.
